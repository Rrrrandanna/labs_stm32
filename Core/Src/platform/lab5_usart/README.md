# Лабораторная работа №5. Управление светодиодом через USART

В этом проекте для работы с USART подготовлены готовые функции. Вам не нужно разбираться в регистрах или HAL – просто используйте их по инструкции. Светодиодом на плате можно управлять, отправляя команды из терминала.

## Доступные функции

| Функция | Описание |
| :--- | :--- |
| `plt_uart_init()` | Инициализирует USART, выводит меню и запускает прием команд |
| `plt_uart_send(const char* data)` | Отправляет строку текста в терминал |
| `plt_uart_print(const char* format, ...)` | Отправляет форматированную строку |
| `plt_uart_is_available()` | Проверка был ли прием по UART |
| `plt_uart_proccess()` | Обработка полученной команды по UART |

> **Важно:** Прием команд происходит в фоновом режиме (в прерывании). Готовый колбэк `HAL_UART_RxCpltCallback()` выставляет флаг приема команды. Вам нужно вызвать `plt_uart_init()` в начале программы и в цикле вызывать plt_uart_proccess().

## Задание 1

1.  В функции `main()` после инициализации периферии вызовите `plt_uart_init()`;
2.  Запустите программу на отладочной плате и откройте терминал (терминал: скорость 115200, Flow Control = None);
3.  Убедитесь, что в терминале появилось приглашение и меню;
4.  Отправьте в терминал команду `1` – светодиод должен загореться, команду `0` – погаснуть;
5.  При отправке любой другой клавиши терминал должен выдать сообщение об ошибке `[ERROR] Use 1 or 0 only`.

## Задание 2 (Коды ошибок)

Добавьте в проект отправку числовых кодов ошибок: это будет полезно для автоматизации тестирования.

1.  Изучите функцию `send_response(uint8_t code, const char* msg)` в примере;
2.  Измените код колбэка так, чтобы при успешном включении/выключении LED отправлялся код `0`, а при ошибке (неизвестная команда) – код `1`.
3.  Пример работы терминала должен выглядеть так:
    ```text
    > 1
    [CODE: 0] LED ON
    > 0
    [CODE: 0] LED OFF
    > 5
    [CODE: 1] Use only 1 or 0
    ```
