# Лабораторная работа: Порт ввода-вывода GPIO

В этом проекте Вам предлагается освоить порты ввода вывода GPIO.

## Задание
Используя функцию-обертку
```c
void plt_gpio_toggle(void)
```
реализуйте управление ярскостью светодиода по следующему алгоритму:
- яркость плавно возрастает до максимального значения
- яркость плавно уменьшается до минимального значения
- процесс повторяется циклически

`Для реализации управления яркостью используйте метод ШИМ (PWM) изменяя соотношение времени работы светодиода к времени его простоя (выключенное состояние)`
